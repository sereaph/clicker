# Clicker

**Auto-click tool** — a simple Python script that simulates mouse clicks at timed intervals.

## Usage

```
python main.py
```

Customize the click interval and button in `main.py`.

## Files

- `main.py`   — Main auto-clicker script
- `3.bat`     — Windows batch shortcut to run the clicker
- `log.txt`   — Click history log (auto-generated)

## Requirements

- Python 3.x
- Windows (uses `win32api` / `pyautogui`)

*Originally created for automating repetitive mouse actions.*
